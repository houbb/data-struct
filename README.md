# 数据结构

[data-struct](https://github.com/houbb/data-struct) 用于记录学习 jdk 的设计原理。

为自己编写框架提供参考，后续提供更加丰富的数据结构支持。

# 快速开始

## 准备

jdk 1.7+

## maven 引入

```xml
<dependency>
    <groupId>com.github.houbb</groupId>
    <artifactId>data-struct</artifactId>
    <version>0.0.1</version>
</dependency>
```
